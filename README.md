Parcial: Lista de Compras Inteligente
Aplicación móvil desarrollada en React Native (Expo) para el parcial 1 de la materia.

Opción Elegida
Lista de compras: Permite gestionar productos a comprar de manera local, manteniendo la sesión del usuario mediante un login simple simulado.

Cómo ejecutar la app
Clonar este repositorio en tu computadora.
Abrir una terminal en la carpeta del proyecto y ejecutar el siguiente comando para instalar las dependencias:
npm install
Se recomienda abrir el proyecto en el navegador (Android Studio puede presentar fallas) abriendo el proyecto en Visual Studio Code, abriendo la consola y ejecutando: npx expo start pulsar w navegar por la aplicación
Demo subida a YouTube: https://youtu.be/4Bk2Hpz_sCY
